Quiz Birds

All bird images are from wikipedia, and have been downsized.

The quiz is generated from the birds found as directories in static/birds.

If you want to add a new bird, create a directory underneath static/birds using the name of the bird. Put in a file prefixed with img_ for the image of the bird, a file prefixed with ruf_ for the audio of the bird’s call and a file prefixed with son_ for a sonogram image of the bird’s call.
